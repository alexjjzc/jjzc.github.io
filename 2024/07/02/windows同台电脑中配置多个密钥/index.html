1.背景

想要在一台主机中配置多个仓库连接时，需要不同的 ssh 密钥来管理，比如：

  • Github 仓库：用于自己个人的开发管理
  • Gitlab 仓库：用于公司内部开发管理

各个仓库上传互不影响，该如何处理？

本地环境下已安装部署完 Git 以及Node，此处不再多赘述。
